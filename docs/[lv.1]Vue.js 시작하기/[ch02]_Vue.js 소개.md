# MVVM 모델에서의 Vue



## Vue는 무엇인가?

* View : 브라우저에서 사용자에게 보여지는 화면
* ViewModel
  * DOM Listeners : View에서 발생하는 이벤트(ex. 클릭, 키보드 입력)를 청취하고 있는다.
    * JS에 있는 데이터를 변경해주거나 JS에 지정한 특정 로직을 실행하게 된다.
  * Data Bindings : JS 데이터가 변경되면 Data Bindings를 타고 화면에 반영하게 된다.

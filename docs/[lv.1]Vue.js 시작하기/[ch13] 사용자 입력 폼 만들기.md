# 프로젝트 생성 및 마크업 작업

```shell
# 프로젝트 생성
vue create vue-form

# 프로젝트 디렉토리로 이동
cd vue-form
# 서버 실행
npm run serve
```

```
1. HelloWorld.vue 삭제
2. App.vue 파일 내용 전부 삭제 후, `vue` 입력해서 `tab`으로 기본 템플릿 자동완성
3. App.vue에 div 태그로 텍스트 출력
```
